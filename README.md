# React Card Gallery

React Card Gallery is a front‑end project designed to demonstrate component architecture, state management, and UI behavior in a modern React environment. It showcases how to build reusable components, manage paginated data, and create interactive UI elements without relying on external UI libraries.

Each card is generated from a predefined dataset and assigned a unique HSL color at runtime. Users can browse through pages of cards, click any card to view details in a modal, and navigate using a custom pagination component. The layout is fully responsive, using CSS Grid and simple styling to keep the interface clean and polished.
